# Django To-Do Application

A simple Django-based To-Do application with user registration, 
login, authentication and personal task management. 
Each user can create, update and delete their own tasks.

## ‚ú® Features
- User registration and login
- Authentication and session-based access
- Create, update and delete personal tasks
- Task priority (low / medium / high)
- Task status (to do / in progress / done)
- Deadlines
- Admin panel for managing users and tasks

## üöÄ Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/todo-django.git
cd todo-django

### 2. Create a virtual environment

python -m venv venv

### 3. Activate the virtual environment

venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt

### 5. Apply migrations

python manage.py migrate

### 6. Run the server

python manage.py runserver

### 7. Open in browser

http://127.0.0.1:8000/
